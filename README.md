Vite project -

NOTES
- files in `/public` get added into the build in `/dist`
  - audio
  - images 
  - etc.

## 0. don't forget to install Node packages

```bash
npm install
```


## 1. run dev version

```bash
npm run dev
```

## 2. build dist(ribution) version in `dist`

```bash
npm run build 
```

and preview the build with:
```bash
 npm run preview
```
